Enhanced. This is a super weeder report that allows you limit by the beginning of a call number. Enter the beginning of a call number, followed by a percent sign. Examples: 0% (for 000s) or J 0% (for J 000s) or LP F% (for LP fiction) in the first box, the change the library name dropdown to your library and click Run. #weeding

<p><span style="background-color: darkred; color: white">Everything this report can do can now be done with report 2731, therefore, this report is scheduled for deletion on March 1, 2018.</p>

----------

SELECT CONCAT(  '<a href=\"/cgi-bin/koha/catalogue/detail.pl?biblionumber=',b.biblionumber,'\" target="_blank">',b.biblionumber,'</a>' ) as 'Bib Number', i.holdingbranch, i.barcode, i.itemcallnumber, b.title, t.publicationyear as 'Copyright', i.dateaccessioned as 'Accessioned', i.itype, i.ccode, i.location, i.issues, i.renewals, (IFNULL(i.issues,0)+IFNULL(i.renewals,0)) as Total_Circ, i.datelastborrowed, i.itemlost, i.onloan, i.damaged, i.itemnotes FROM items i LEFT JOIN biblio b USING (biblionumber) LEFT JOIN biblioitems t USING(biblionumber) WHERE i.homebranch=<<Select your Library|branches>> AND i.itemcallnumber LIKE <<Enter call number beginning, with % after (Ex. 0% or LP F% or ER%)>> ORDER BY i.itemcallnumber
