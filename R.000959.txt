Shelf list for a section.  Uses Call Number to create a smaller shelf list to use in the DLA sorted by author and call number. Choose branch & enter beginning of call number with percentage sign.

----------

SELECT items.barcode,
  items.homebranch,
  items.itemcallnumber,
  biblio.title,
  biblio.author
FROM items JOIN biblio USING(biblionumber)
WHERE items.homebranch = <<branch|branches>> AND
  items.itemcallnumber LIKE Concat(<<enter beginning of call number with % sign (ex. BB%)>> , "%")
ORDER BY biblio.author,
  items.itemcallnumber asc
