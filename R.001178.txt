Our best attempt to find what should be on the shelf.  Enhanced to allow you to select Branch and Collection Code and Shelving Location.

<p><span style="background-color: darkred; color: white">Everything this report does can be done by report 2731.  Unless someone contacts nexpresshelp@nekls.org with further information, this report is scheduled for deletion on September 1, 2018.</p>

----------

SELECT
  items.onloan,
  items.itemcallnumber,
  biblio.title,
  items.barcode
FROM
  items
  JOIN biblio ON items.biblionumber = biblio.biblionumber
WHERE
  items.homebranch = <<Pick your branch|branches>> AND
  items.ccode = <<Pick your collection code|CCODE>> AND
  items.location = <<location|LOC>> AND
  items.damaged = 0 AND
  items.itemlost = 0 AND
  items.notforloan = 0
ORDER BY
  items.onloan,
  items.itemcallnumber
